# java-cli-maven-surefire-spring-findbugs-jacoco-pmd-serenity-cucumber-junit5-car-data

## Description
A POC for maven app using junit5
serenity, and cucumber framework
 with pmd,
jacoco, findbugs and surefire plugin.

## Tech stack
- java
- maven
	- spring
  - junit5
  - surefire
  - jacoco
  - pmd
  - findbugs
  - cucumber
	- serenity



## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
